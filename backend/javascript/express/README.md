## express

### Environment
- node v19.1.0
- nodenv v1.4.1
- docker v20.10.22

### Install node modules
```
npm i
```

### Run server
```
npm run start
```

### Run server by docker
```
docker-compose up
```

### curl

ping
```
curl http://localhost:3000/api/v1/ping
```
deep
```
curl http://localhost:3000/api/v1/deep
```

### mongo
